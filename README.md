# Сайт «Семь чудес света» (seven-wonders)
Личный проект по верстке, выполненный на HTML/CSS. Сайт рассказывает о семи чудесах света. Каждому чуду света посвящена отдельная страница.
В проекте реализованы:
- семантическая верстка;
- адаптация под компьютер, планшет и телефон;
- единый стиль на всех страницах.
